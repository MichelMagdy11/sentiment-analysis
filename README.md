# 🧠 Sentiment Analysis using LSTM & GRU

## 📌 Project Overview

This project focuses on **Sentiment Analysis**, a Natural Language Processing (NLP) task used to determine whether a piece of text expresses a **positive or negative sentiment**.

In this project, I implemented deep learning models using **LSTM** and **GRU** architectures to classify text data based on sentiment.

---

## 🚀 Features

* Text preprocessing (cleaning, tokenization, padding)
* Implementation of **LSTM** and **GRU** models
* Model training and evaluation
* Performance comparison between models
* Visualization of training results

---

## 🛠️ Technologies Used

* Python 🐍
* TensorFlow / Keras
* NumPy & Pandas
* Matplotlib / Seaborn

---

## 🧪 Model Architecture

* Embedding Layer
* LSTM / GRU Layers
* Dense Layers
* Output Layer (Sigmoid for binary classification)

---

## 📊 Results

* The models were evaluated using accuracy and loss metrics
* LSTM and GRU showed strong performance in capturing sequential dependencies in text
* Comparison highlighted differences in performance and training time
